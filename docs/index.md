# 前言

​		作为一个程序员，我们肯定是更倾向于用代码来实现各种逻辑的，这也是本书的初衷。

​		现如今，市面上大部分教程要么，都是基于蓝图构建的，要么是基于UE4C++版本的教程。对于蓝图教程，作为程序员用起来感觉缺点什么，对于UE4C++教程，写起来可能会遇到UE5新的特性，有一些更好的选择，比如Enhanced Input增强输入之类的。笔者之前学习的时候，除了官网的文档以外，鲜有具体的UE5C++教程书籍，故心血来潮便有了这本书。

​		本书是以制作一款多人第三人称射击游戏为主要内容，尽可能将逻辑放到C++上的一本书。笔者使用的是UE5.4.4版本，建议以此版本为准，并将UE编辑器语言切换为英文（方便查看对应属性和编写代码）。笔者参考Lyra用C++代码进行整个项目的讲解和制作。
